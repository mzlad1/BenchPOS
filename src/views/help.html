Frequently Asked Questions

Can I process a return if the customer doesn't have their receipt? +

Yes, you can look up the transaction using:

  • Customer name or phone number
  • Payment method details
  • Date of purchase with approximate amount

Use the search filters in the transaction history to narrow down potential matches.

Can I process returns after the return policy period? +

The system will display a warning if you attempt to process a return outside the configured return period. By default, returns after:

  • 30 days require manager approval
  • 90 days require administrator approval

These timeframes can be customized in your system settings.

How do I handle exchanges instead of returns? +

For exchanges:

  1. Process the return as normal
  2. Create a new sale for the replacement item
  3. In the notes section of both transactions, reference the related return/new purchase

For even exchanges (same price), you can check the "Exchange" option during the return process, which will generate a credit note that can be applied immediately to a new sale.

Frequently Asked Questions

Can I add product images to my inventory? +

Yes, you can add product images after creating the basic product entry:

  1. In the product list, click on a product name to open its detail view
  2. In the Media tab, click "Add Image"
  3. Select an image file from your computer (supported formats: JPG, PNG, GIF)
  4. Click "Upload" to add the image to the product
How do I add products with variations (size, color, etc.)? +

For products with variations, you have two options:

  • Create separate products: Make individual entries for each variation with unique Bar Codes (e.g., TSHIRT-S, TSHIRT-M, TSHIRT-L)
  • Use the Variants feature: Create a parent product, then add variants with specific attributes. Access this by creating a product, then clicking "Add Variants" in the product detail view.

When you delete a product:

  • The product is removed from active inventory
  • Past transactions involving this product remain intact for reporting purposes
  • The product SKU becomes available for reuse

If you want to keep the product but prevent new sales, consider setting the stock to zero instead of deleting it.
